# JavaScript Mini Projects

This repository contains small projects built while learning core JavaScript concepts such as DOM manipulation, event handling, and API integration.

## Projects

### 1. Stone Paper Scissors
A simple browser game where the user plays against the computer.

Concepts used:
- DOM manipulation
- Event listeners
- Random number generation

### 2. Tic Tac Toe
A classic two-player game implemented using JavaScript.

Concepts used:
- Game logic
- DOM updates
- Event handling

### 3. Currency Converter
A currency converter that fetches real-time exchange rates using an API.

Concepts used:
- Fetch API
- Async/Await
- API integration

## Technologies Used

- HTML
- CSS
- JavaScript
